如何使用

  1. 获取API密钥： 联系管理员获取您专属的API密钥
  2. 配置环境： 设置 ANTHROPIC_API_KEY 和 ANTHROPIC_BASE_URL
  3. 开始使用： 像使用官方API一样发送请求到中转站
  4. 监控使用： 通过用户面板查看使用情况和统计数据

配置示例

# 环境变量配置
export ANTHROPIC_API_KEY="sk-proxy-xxxxxxxxxxxx"
export ANTHROPIC_BASE_URL="http://relay-station:8000"

# Python 代码示例
import anthropic

client = anthropic.Anthropic(
    api_key="sk-proxy-xxxxxxxxxxxx",
    base_url="http://relay-station:8000"
)

response = client.messages.create(
    model="claude-3-5-sonnet-20241022",
    max_tokens=1000,
    messages=[{
        "role": "user", 
        "content": "Hello, Claude!"
    }]
)
